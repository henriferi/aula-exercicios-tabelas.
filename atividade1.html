Table- Tag usada para criar uma tabela

TH- Tag usada dentro de uma tabela para criar um cabelhaço

TR- tag usada para criar linhas na tabela

TD- Tag - usada para preencher células da tabela

T-HEAD - indica o cabeçalho da tabela

TBODY- Indica o corpo da tabela

TFOOT- Indica o rodapé da tabela

Align- É usado para mudar a posição de textos ou imagem/video etc no texto exemplo (Align="center") para o texto ficar no centro.

Border- usado para por borda na tabela

Caption - usado para criar um titulo para a tabela

Rowspan - usado para fazer uma célula ocupar mais de uma linha na tabela

Colspan - usado para fazer uma célula ocupar mais de uma coluna na tabela

Width - Define a largura de uma imagem ou video.
